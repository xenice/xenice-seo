# WordPress插件: xenice-seo
> 简单的WordPress SEO插件，为网站首页、文章页、分类页、标签页、自定义文章类型页面等添加SEO标题、SEO关键词、SEO元描述，支持网站地图、中英文切换。  

### 插件特色

- 自定义SEO标题、关键词和元描述
- 文章摘要作为元描述（可选）
- 文章标签作为关键词（可选）
- 标签&分类描述作为分类元描述（可选）
- 网站地图（可选
